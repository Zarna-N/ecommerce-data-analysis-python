# ecommerce-data-analysis-python
This project focuses on analyzing e-commerce sales data using Python to uncover meaningful business insights. It involves data cleaning, preprocessing, and visualization to identify sales trends, top-performing products, and customer behavior patterns. The goal is to support data-driven decision-making for business growth.
🔍 Key Insights
- Electronics category generated highest revenue
- Peak sales observed during festive months
- UPI and Credit Card are most preferred payment methods
